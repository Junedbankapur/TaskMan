# Task Manager App

A task management web application built using React and Vite. This project was developed as part of my final React coursework and enhanced with custom features.

## Features

* Add new tasks
* Delete tasks
* Mark tasks as completed
* Clean and responsive UI

## Tech Stack

* React.js
* Vite
* JavaScript
* CSS

## Learning Outcomes

* Component-based architecture
* State management in React
* Routing and UI structuring
* Git & GitHub workflow

## Note

This project was initially guided by coursework and further enhanced by me.
